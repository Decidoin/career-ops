# Mode: Pivot-Check

Triggered by: `/career-ops pivot-check`

Quick assessment tool: paste any opportunity and get a focused pivot alignment score.

## Workflow

1. Read the opportunity (JD, description, or URL)
2. Score ONLY on pivot alignment dimensions:

| Check | Question | Score |
|-------|----------|-------|
| **AI Agent Connection** | Does this role involve AI agent systems, multi-agent coordination, or autonomous systems? | 0-10 |
| **Mechanism Design** | Does this role involve designing incentive systems, auctions, scoring, or coordination rules? | 0-10 |
| **Transferability** | Can Taani's blockchain governance experience be directly applied? | 0-10 |
| **Narrative Fit** | Does taking this role make the pivot story stronger or weaker? | 0-10 |
| **Learning Velocity** | Will this role teach skills needed for the AI agent governance future? | 0-10 |

3. Output a single verdict:
- **🟢 Advances pivot** (avg 7+): This role moves you closer to where you want to be
- **🟡 Neutral** (avg 4-6): Doesn't hurt, doesn't help much
- **🔴 Regression** (avg <4): Taking this role would make the pivot harder to explain
